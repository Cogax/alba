Alba

A humble library of helpers and a Trie-based router for OWIN applications


Find Out More Get started!

Polyglot Persistence for .Net Systems using the Rock Solid Postgresql Database

OWIN Helpers

Alba is your one stop source for all manner of OWIN helpers to make using the "mystery meat" OWIN environment dictionary easier and more discoverable. Work with HTTP headers, cookies, parse the request body, logic around url's, and write content back to the OWIN response. Alba can even help with custom content negotiation selection.

View details »

Trie-based Router

Alba provides an HTTP router based off of the proven Trie algorithm for more optimized routing performance. Alba's router can be used to simplify OWIN applications by acting as a front controller to your actual route handlers.

View details »

Middleware

Alba comes out of the box with an optimized static file middleware implementation and an "html head injection" middleware stategy that can be useful in many automated testing scenarios.
